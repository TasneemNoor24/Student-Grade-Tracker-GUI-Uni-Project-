# Student Grade Tracker

## Overview

Student Grade Tracker is a Java-based application designed to manage student records and grades efficiently. The application provides a simple graphical user interface (GUI) for adding students, entering marks, calculating grades, and viewing academic performance.

## Features

* Add student records
* Enter and update student marks
* Calculate grades automatically
* Display student information in a table
* User-friendly Java Swing GUI
* Easy and efficient record management

## Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)
* NetBeans IDE

## How to Run

1. Open the project in NetBeans IDE.
2. Build the project.
3. Run the main Java file.
4. Use the application to add student records and manage grades.

## Project Structure

* `src/` – Java source code
* `nbproject/` – NetBeans project configuration files
* `README.md` – Project documentation

## Purpose

This project was developed to demonstrate Java programming concepts, Object-Oriented Programming (OOP), and Graphical User Interface (GUI) development.
